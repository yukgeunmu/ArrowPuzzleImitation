# Arrow Puzzle

Unity 기반의 화살 퍼즐 게임입니다.

플레이어는 각 화살을 선택하여 이동시키고, 모든 화살을 보드 밖으로 탈출시키는 것을 목표로 합니다.

본 프로젝트는 단순한 스테이지 클리어 방식이 아닌, **무한 자동 생성 퍼즐 시스템**을 중심으로 개발되었습니다.

---

## 🎮 게임 소개

Arrow Puzzle은 화살의 방향과 배치를 분석하여 모든 화살을 보드 밖으로 탈출시키는 퍼즐 게임입니다.

* 화살 선택 시 해당 방향으로 이동
* 장애물 또는 다른 화살에 막히면 정지
* 보드 밖으로 나가면 제거
* 모든 화살 제거 시 스테이지 클리어

---

## ✨ 주요 기능

### 1. 무한 자동 생성 스테이지

기존의 수작업 스테이지 제작 방식 대신 역생성(Reverse Generation) 기반 알고리즘을 구현하였습니다.

* 플레이 가능한 퍼즐 자동 생성
* 매 스테이지마다 새로운 배치 생성
* 난이도에 따른 퍼즐 크기 조절
* 별도 스테이지 데이터 제작 불필요

### 2. 난이도 시스템

플레이어가 원하는 난이도를 선택할 수 있습니다.

| 난이도    | 보드 크기   |
| ------ | ------- |
| Easy   | 6 x 6   |
| Normal | 8 x 8   |
| Hard   | 10 x 10 |
| Expert | 15 x 15 |
| Master | 20 x 20 |

### 3. 힌트 시스템

초기 구현은 BFS Solver를 사용하여 최적 해답을 계산하였습니다.

그러나 대형 퍼즐에서 성능 문제가 발생하여 다음과 같이 개선하였습니다.

* 즉시 탈출 가능한 화살 우선 탐색
* Solver 호출 최소화
* 모바일 환경 대응

### 4. 카메라 시스템

* 핀치 줌
* 드래그 이동
* 자동 보드 맞춤(Fit To Grid)
* 스테이지 변경 시 카메라 초기화

### 5. 기록 시스템

난이도별 최고 기록 저장

* 클리어 시간 측정
* Best Time 저장
* PlayerPrefs 기반 로컬 저장

## ⚙️ 자동 생성 알고리즘

### 초기 방식

BFS Solver를 이용하여 생성된 퍼즐의 정답 여부를 검증

문제점

* 화살 수 증가 시 탐색 비용 급증
* 20x20 퍼즐에서 실시간 생성 어려움

### 개선 방식

역생성(Reverse Generation) 적용

1. 탈출 가능한 화살 생성
2. 내부 공간으로 확장
3. Flow Map 생성
4. 충돌 검증
5. 플레이 가능한 퍼즐 구성

결과

* Solver 의존도 감소
* 생성 속도 향상
* 무한 스테이지 구현 가능
